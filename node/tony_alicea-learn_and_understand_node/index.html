Big word: HTTP

A set of rules (and a format) for data being transferred on the web.
Stands for 'Hypertext Transfer Protocol'. It's a format (of various) defining data being transfrerred via TCP/IP

Big word: MIME type

A standard for specifying the type of data being sent.
Stands for 'Multipurpose Internet Main Extensions'. Examples: application/json, text/html, image/jpeg.
