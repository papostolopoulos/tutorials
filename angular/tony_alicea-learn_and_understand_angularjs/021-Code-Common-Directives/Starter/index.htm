Directives that were presented in this lecture

  • ng-if. If statement for true or false
  • ng-show. For showing page components based on particular conditions
  • ng-hide. For hiding page components based on particular conditions
  • ng-class. For adjusting classes
  • ng-repeat. Similar to the for loop
  • ng-click. All the different events that are typically used in Vanilla JS can also be used through AngularJS directives.
  • ng-cloak. For not displaying the elements inside the brackets {{}} until the page fully loads
