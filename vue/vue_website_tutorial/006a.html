Controlling Reusable Elements with key

The key attribute in the input tag is the one that defines that the input tags are separate from each other. In some cases we want the same tags to just have different content while in some other cases we want the tags to be completely differet.

v-show

Another option for conditionally displaying an element is the v-show directive. The usage is largely the same.
The difference is that an element with v-show will always be rendered and remain in the DOM; v-show simply toggles the display CSS property of the element.
