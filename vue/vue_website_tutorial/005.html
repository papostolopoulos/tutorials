5 - Conditional Rendering

Definition

An If - else statement can be used as a Vue directive, same as v-bind directive is used for adding attributes in the html code. v-if, v-else-if and a v-else directives can be used to render different parts of html based on if conditions are satisfied or not.

Example

The following example renders different div tags based on if a condition is met or not. This condition has been set with a random number that either displays the div tag either it displays its alternative. Refresh the page a couple of times to see how there can be a toggle between the one and the other tag. 

          <div v-if="Math.random() > 0.5">
            Now you see me
          </div>
          <div v-else>
            Now you don't
          </div>
Now you see me
Now you don't

Toggle between different placeholders and labels with the v-if directive.

Controlling Reusable Elements with key

The key attribute in the input tag is the one that defines that the input tags are separate from each other. In some cases we want the same tags to just have different content while in some other cases we want the tags to be completely differet.

v-show

Another option for conditionally displaying an element is the v-show directive. The usage is largely the same.
The difference is that an element with v-show will always be rendered and remain in the DOM; v-show simply toggles the display CSS property of the element.
