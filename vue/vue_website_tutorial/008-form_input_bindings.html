8 - Form Input and Bindings

Basic usage

The v-model directive is a syntactical sugar way for updating data on user input events. This two way data binding directive allows the use of input, textarea and select tags.

v-model ignores the initial value, selected or checked attributes on form elements. It always treats the Vue instance data as the source of truth. For an initial value, it is better if we use the value of the particular data property that is bound to the v-model directive.

Examples

Text

The text from the input is:
{{input}}


<p>The text from the <code>input</code> is: { {input} }</p> <input type="text" name="" value="" v-model="input">
Multiline text

The text from the textarea is: {{textArea}}


<p style="white-space: pre;"> The text from the <code>textarea</code> is: { {textArea} } </p> <textarea rows="2" cols="80" v-model="textArea"></textarea>
Checkbox

The box is checked: {{checkBox}}

<p> The box is checked: {{checkBox}} <input type="checkbox" name="" value="" v-model="checkBox"> </p>
Multiple checkboxes

Multiple check boxes array: {{multiCheckBoxes}}

<input type="checkbox" name="Giorgos" id="Giorgos" value="Giorgos" v-model="multiCheckBoxes"> <label for="Giorgos">Giorgos</label> <input type="checkbox" name="Olga" id="Olga" value="Olga" v-model="multiCheckBoxes"> <label for="Olga">Olga</label> <input type="checkbox" name="Stam" id="Stam" value="Stam" v-model="multiCheckBoxes"> <label for="Stam">Stam</label>
Radio Buttons

Radio Buttons selection: {{radioButton}}

<input type="radio" name="" value="Giannis" id="Giannis" v-model="radioButton"> <label for="Giannis">Giannis</label> <input type="radio" name="" value="Lefteris" id="Lefteris" v-model="radioButton"> <label for="Lefteris">Lefteris</label> <input type="radio" name="" value="Evangelos" id="Evangelos" v-model="radioButton"> <label for="Evangelos">Evangelos</label>
Select

Select option: {{select}}

<select class="" name="" v-model="select"> <option disabled value="">Select an option</option> <option value="Maria">Maria</option> <option value="Sofia">Sofia</option> <option value="Georgia">Georgia</option> </select>
Select multiple

Selected options: {{selectMultiple}}

<select class="" name="" v-model="selectMultiple" multiple> <option value="Tasia">Tasia</option> <option value="Sakis">Sakis</option> <option value="Makis">Makis</option> <option value="Takis">Takis</option> </select>
Dynamic options rendered with v-for

Dynamic option selection: {{dynamicSelection}}

<select class="" name="" v-model="dynamicSelection"> <option disabled value="" selected>Select an option</option> <option v-for="option in options" :value="option.value">{ {option.option} }</option> </select>
