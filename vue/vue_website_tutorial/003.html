1 - Interpolations

#Text

The most basic form of data binding is text interpolation using the “Mustache” syntax (double curly braces):

Example

Message: {{message}}
The text after the colon will never change: {{message}}.
The v-once directive is used so even if the message property in the data property changes, v-once keeps it unchanged.

#html

The double mustaches interprets the data as plain text, not HTML. In order to output real HTML, you will need to use the v-html directive.

Example

Using mustaches:
{{ rawHtml }}

Attributes

Mustaches cannot be used inside HTML attributes, instead use a v-bind directive:

Example

<div v-bind:id="dynamicId"></div>

When it comes to boolean attributes, v-bind works in a different way. For example: <button v-bind:disabled="isButtonDisabled">Button</button>, If the isButtonDisabled has a value of null, false, undefined, then the value will not be included in the in the button element
