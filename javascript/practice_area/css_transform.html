1 Transition width

Transition of 2 seconds with the change of the width.


      .transition-width {
        transition: width 2s;
      }

      .transition-width:hover {
        width: 200px;
      }
hover over me

2. Transition border

Transition of 2 seconds with the change of the border.


      .transition-border {
        transition: border 2s;
      }

      .transition-border:hover {
        border: 6px solid green;
      }
hover over me

3. Transition width - height

Transition of 2 seconds for changing the width and 4 seconds for changing the height (don't forget the comma).


      .transition-width-height{
        transition: width 4s, height 2s;
      }

      .transition-width-height:hover {
        width: 200px;
        height: 50px;
      }
hover over me
