translate() function

Means moving the element to a different location. We can use translateX for moving them horizontally, translateY for moving them vertically and both at the same time when we use translate().

.box5 { width: 200px; height: 200px; background: #475; -ms-transform: translate(50px, 100px); -webkit-transform: translate(50px, 100px); transform: translate(50px, 100px); }

scale() function

Means making the element larger or smaller. Similar to translate(), scaleX for horizontal and scaleY() for vertical modifications.

.box6 { width: 200px; height: 200px; background: #475; -ms-transform: scale(.5, .1); -webkit-transform: scale(.5, .1); transform: scale(.5, .1); }
