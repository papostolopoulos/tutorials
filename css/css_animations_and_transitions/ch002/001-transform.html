translate() function

Means moving the element to a different location. We can use translateX for moving them horizontally, translateY for moving them vertically and both at the same time when we use translate().

.box5 { width: 200px; height: 200px; background: #475; -ms-transform: translate(50px, 100px); -webkit-transform: translate(50px, 100px); transform: translate(50px, 100px); }

scale() function

Means making the element larger or smaller. Similar to translate(), scaleX for horizontal and scaleY() for vertical modifications. Values taken are positive numbers. Decimals less than 1.0 mean a percentage of the original size while values larger than 1.0 mean a size more than 100% of the original size.

.box6 { width: 200px; height: 200px; background: #475; -ms-transform: scale(.5, .1); -webkit-transform: scale(.5, .1); transform: scale(.5, .1); }

rotate() function

Means we rotate objects in a two dimentional space. The rotation happens around a single point which is set with the transform-origin property. If the transform-origin property is not set to a different value, the rotation happens from the center of the element. Values that can be used are degrees (deg) or radians (rad).

.box1 { width: 200px; height: 200px; background: #475; -ms-transform: rotate(45deg); -webkit-transform: rotate(45deg); transform: rotate(45deg); }

skew() function

This property is used when we want an object to be distorted. skewX() is used when we want to distort an element horizontaly, while skewY() is used when we want to distort the element vertically. The values that we use are the same like the rotate() function (degrees and radians).

.box7 { width: 200px; height: 200px; background: #475; -ms-transform: skewX(10deg) skewY(30deg); -webkit-transform: skewX(10deg) skewY(30deg); transform: skewX(10deg) skewY(30deg); }

matrix

A matrix is an array of numbers that use matrix math for adding, subtracting, multiplying and dividing matrices.
